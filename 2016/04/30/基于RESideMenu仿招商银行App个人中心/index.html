基于RESideMenu快速实现类手机QQ侧划交互

发表于 2016-04-30   |   分类于 iOS   |     |   次阅读

框架选择

汉堡菜单：你同意或者否定，它就在那里。

  当你基于这样一个兴趣或者需求，想要来一个像手Q一样的侧滑菜单界面，最快的莫过于采用第三方类库了。找星星最多的，RESideMenu映入了我们的眼睑。那还等什么，开工。
  我们来看QQ的交互效果是怎么样的。首页点击“头像”菜单弹出左侧导航页面， 我们以“我的文件”为例。 点击我的文件进入我的文件viewController，此时从“我的文件”页面返回。What？居然直接回到了主页。如下图所示：

  从导航栈来说，那必定是从主页推出的左侧导航栏的子栏目页面，具体是怎样的呢？

  观察市面上的App，招商银行、知乎日报的效果和手机QQ的大同小异。以RESideMenu为例，本身有当前视图、左侧视图、右侧视图三个视图设置

  • (void)commonInit
    {
    //———————–//
    //_parallaxEnabled = YES;
    _parallaxEnabled = NO;

    _contentViewInLandscapeOffsetCenterX = 30.f;
    _contentViewInPortraitOffsetCenterX = 30.f;
    _contentViewScaleValue = 0.7f;
    }
    ```
    30代表主页界面在当前侧面视图显示的宽度，0.7代表主页视图缩放的大小。 完全不缩放则scaleValue值为1。在原样大小下，要调整主页显示的宽度来使界面更好看，具体的数值可以慢慢探索。

RESideMenu默认侧栏页面有重力感应效果，此处我们将_parallaxEnabled赋值为NO，实际可以根据爱好及需求进行调整。
至此两个问题的解决对RESideMenu的应用算是有了一定的了解。当然Demo也是有的，客官都看到这了顺手点赞支持一下可好。
