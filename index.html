关于运动、音乐与读书

我们的目标是快乐。–《幸福课》

运动

  坊间流传程序员必备的书籍有《颈椎病康复指南》、《高血压预防》与《活着》等，看的时候莞尔一笑，细细想来确实离我们很近。偶有英年早逝的同仁先一步去追寻0与1的终极奥义，活着的我们不胜唏嘘，身体还是第一位的。身体机能从25岁开始需要我们有意识的保持才能像少年一样迎着朝阳飞驰。每周适量的运动不仅是是对多巴胺分泌的刺激，也是为了增强免疫系统，为我们更(da)好(jia)编(jie)程(she)创造了良好的条件。有一两项运动爱好的你想必都没有这个烦恼，对于体育盲，简单而有效的方式就是跑步了。
  一般而言每周最少要跑步三次，每次30分钟以上的运动量才能维持身心的愉悦。不论繁忙的时候还是懒癌发作，坚持运动才是必要的。坚持这个词有抵触的情绪在，不过刚开始跑步跑个两三公里都是坚持，天热腿困心烦躁，跑够30分钟可是个不小的挑战，可以不想一个月的目标，就想想今天这次跑够时间再说，不用几公里，跑慢点跑够三四十分钟就好。

DIY你的iOS应用启动图片

  通用的iPhone启动图片有四种尺寸，分别为：

  • iPhoen4s/4(@2x) : 640 x 960
  • iPhoen5s/5(@2x) : 640 x 1136
  • iPhone 6(@2x) : 750 x 1334
  • iPhone 6 plus (@3x) : 1242 x 2208

正文

  我们开发拿到的图片素材基本是基于iPhone6屏幕尺寸或者1920x1080像素这样的尺寸。如何在有限的一张素材图的情况下配齐我们的四种机型启动图呢？

  简单就是极致，Mac自带的图片编辑工具就可以搞定这一切。

  1. 首先选中我们要处理的图片，按空格进入展示框。
  2. 在新弹出的窗口中选择“Open with Preview/预览”
  3. 在顶栏的菜单栏选择“Tools/工具”
  4. 选择“Adjust Size/调整大小”

至此我们会面对如下展示的界面

这里面需要我们调整的有Width, Height, Resolution 和 Scale roportionally。其中Scale roprotionally取消勾选， 分辨率(Resolution)设置为72，再需要调整的就是宽跟高的像素以符合我们的机型配置。

iOS视频录制压缩

  所见所分享。记录眼前的美好事物并通过我们的App进行展示。作为开发者，满足用户多长时间的拍摄时间为宜，15秒还是像微博取消了140字限制一样给个1分钟的拍摄时间。理论上可以长久录制视频，但克制才是真爱，如何为1分钟的视频压缩优化？

项目应用AVFoundation

捕获视频主要为AVCaptureSession：

  • objc-在iOS上捕获视频
  • 模仿快手应用的视频录制

  基本的视频录制通过以上博文和Demo即可做出，其中：

_captureSession=[[AVCaptureSession alloc]init];
if ([_captureSession canSetSessionPreset:AVCaptureSessionPreset640x480]) {//设置分辨率
    _captureSession.sessionPreset=AVCaptureSessionPreset640x480;
}

  AVCaptureSessionPreset可以设置不同的采样分辨率，此处基于清晰度选择640*480。为什么不选择High?

  动辄上百兆的空间占用，还给你报个Memory received warning！ 选640*480清晰度达到要求，视频采集数据只有不到30M的占用，世界清静了。
