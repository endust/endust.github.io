浅谈公共交通卡刷卡设备

公交卡

公交卡是智能卡的一种，一般称公共交通一卡通。主要应用于城市公共交通的小额支付。我们一般市面上遇见的有大概四类：

  1. IC卡，即常规卡
  2. 具有刷卡功能的NFC手机
  3. 具有刷卡功能的SIM卡手机
  4. 具有刷卡功能的手环等设备
  5. 具有刷卡功能的蓝牙卡设备

在移动互联网时代，出门只带手机成为轻装简行的大众方式。一张小小的公交卡卡片成为制肘，解放双手不带卡呼声越来越高。下四种刷卡方式随着智能手机的普及，已经较多进入我们的视野。

一、NFC刷卡方式

NFC是有Philips、Nokia和Sony主推的一种类似RFID(非接触式射频识别)的短距离无线通信技术标准。NFC刷卡是将NFC终端模拟成一张非接触式智能卡，并可与其读卡器进行交互，发起验证和交易。目前多用硬件的卡模拟即虚拟卡模式，应用在国内一卡通及公交领域。 虚拟卡模式是基于硬件的卡模拟模式，它基于本地安全元件SE(Secure Element)的实现方式。在该模式下，非接触通讯前端由安全模块SE和NFC芯片组成。在手机上，SE有三种集成方案。

在此NFC刷卡指全终端方案，即SE位于手机中。

二、SIM卡刷卡方式

SIM刷卡方式本质是将虚拟卡放入支持NFC功能的终端，即SIM卡中，实现交通卡的各类应用。SIM即为SWP方案，SE位于SIM卡中。手机通过更换SWP-SIM卡，插入支持NFC功能的手机，即可实现刷公交。支持通过手机App充值、查询余额和交易明细查询等。

可以看到实现SWP需要运营商支持，目前中国移动、联通及电信都有各自的SIM卡交通卡，部分地区免费更换，感兴趣的读者可以咨询当地运营商。

三、手环设备刷卡方式

市面上除了出门刷手机还有一种方式就是出门刷手环。一个小小的手环如何支持刷公交功能呢，本质还是一样，将SE及NFC模块置于手环内。有手环厂商每个城市开售对应城市的手环，再通过手机上的App也实现了充值消费等功能。

四、具有刷卡功能的蓝牙卡

什么是蓝牙卡：蓝牙卡也是一种具有公共交通支付功能的设备，可乘坐地铁、公交、便利店购买商品以及过景点闸机等功能。17年飞凡公司推出一款蓝牙卡挂饰产品就是支持西安公共交通等城市的蓝牙卡。蓝牙卡解决了iPhone、安卓用户开卡、充值需要跑线下网店的困扰。蓝牙卡的出现完美解决了移动互联网环境下用户的需求，在家完成某一城市公交卡的开通、充值工作而不需要借助于而外的设备，只需要一部手机，广大非NFC的手机用户也获得了NFC手机用户一样的体验。想想这样的场景：卡里没钱了，不想去网点，不要紧，飞凡帮你搞定！拿出手机，打开飞凡App, 充值，完美，可以愉快的约约约了。

五、NFC在iPhone上的应用现状

与一般安卓手机不同，基于iOS系统的iPhone中的NFC芯片是被苹果“封住”的，或许是为安全因素考虑，苹果公司并未向第三方开发者开放NFC相关的API，目前只支持Apple Pay移动支付，而不能用于两个手机传输数据或者手机与真实公交卡之间的交互，并且iPhone6之前的设备是不支持该功能的。

事未竟然，去年日本东京与苹果公司达成合作，实现了刷iPhone坐地铁。曙光在望。

我们的目标是快乐。–《幸福课》

运动

  坊间流传程序员必备的书籍有《颈椎病康复指南》、《高血压预防》与《活着》等，看的时候莞尔一笑，细细想来确实离我们很近。偶有英年早逝的同仁先一步去追寻0与1的终极奥义，活着的我们不胜唏嘘，身体还是第一位的。身体机能从25岁开始需要我们有意识的保持才能像少年一样迎着朝阳飞驰。每周适量的运动不仅是是对多巴胺分泌的刺激，也是为了增强免疫系统，为我们更(da)好(jia)编(jie)程(she)创造了良好的条件。有一两项运动爱好的你想必都没有这个烦恼，对于体育盲，简单而有效的方式就是跑步了。

  • iPhoen4s/4(@2x) : 640 x 960
  • iPhoen5s/5(@2x) : 640 x 1136
  • iPhone 6(@2x) : 750 x 1334
  • iPhone 6 plus (@3x) : 1242 x 2208

正文

  我们开发拿到的图片素材基本是基于iPhone6屏幕尺寸或者1920x1080像素这样的尺寸。如何在有限的一张素材图的情况下配齐我们的四种机型启动图呢？

  3. 在顶栏的菜单栏选择“Tools/工具”
  4. 选择“Adjust Size/调整大小”

iOS视频录制压缩

  所见所分享。记录眼前的美好事物并通过我们的App进行展示。作为开发者，满足用户多长时间的拍摄时间为宜，15秒还是像微博取消了140字限制一样给个1分钟的拍摄时间。理论上可以长久录制视频，但克制才是真爱，如何为1分钟的视频压缩优化？

项目应用AVFoundation

捕获视频主要为AVCaptureSession：

  • objc-在iOS上捕获视频
  • 模仿快手应用的视频录制

  基本的视频录制通过以上博文和Demo即可做出，其中：

_captureSession=[[AVCaptureSession alloc]init];
if ([_captureSession canSetSessionPreset:AVCaptureSessionPreset640x480]) {//设置分辨率
    _captureSession.sessionPreset=AVCaptureSessionPreset640x480;
}

  AVCaptureSessionPreset可以设置不同的采样分辨率，此处基于清晰度选择640*480。为什么不选择High?

  动辄上百兆的空间占用，还给你报个Memory received warning！ 选640*480清晰度达到要求，视频采集数据只有不到30M的占用，世界清静了。
